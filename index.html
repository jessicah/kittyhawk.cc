Women's Events by Kittyhawk Racing

Currently, only the Fierce Felines is on the calendar. We are also evaluating whether there is interest in a couple other formats, and when the best times might be for putting these on. These include dedicated races for the time trial, criterium, and climbing formats, as well as the chase format.

If you could spare a few minutes, please complete our survey.

All currently scheduled events can be seen at Zwift Events.

You can also join the Kittyhawk Community Events Discord server.

Fierce Felines

Fierce Felines is currently in series 2 of 2023, and the first full 8-week calendar. Unfortunately, it does clash with the Iceni series during September, so numbers will be lighter for the second half.

Please see the full series details, and the published results, which aim to be updated no later than late Sunday.

The current points format is 50, 47, 45, 44, 43, ..., 3, 2, 1, with select events using a points format, that will be described on the series details page, as well as in the Zwift event information.
